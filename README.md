# 🚀 Featured Projects

## 🏦 Credit Risk Scoring – Probability of Default Modeling

Development of an interpretable **credit scoring model** to estimate the probability of default using the UCI Credit Card dataset.

The project explores both **statistical modeling and machine learning approaches** to identify key drivers of default risk.

**Main steps**

• Exploratory Data Analysis (30k observations)  
• Feature engineering and categorical encoding  
• Logistic Regression modeling (interpretable credit scoring)  
• Model evaluation: ROC-AUC, KS statistic, Gini coefficient  
• Lift curve analysis for risk ranking validation  
• Random Forest comparison and feature importance analysis  

**Key Results**

• AUC ≈ 0.71  
• Gini ≈ 0.43  
• KS ≈ 0.37  

The results highlight **payment behaviour variables (PAY_0)** as the strongest predictors of default risk, consistent with credit risk modeling literature.

**Tech:** Python · Pandas · NumPy · Scikit-learn · Statsmodels · Matplotlib  

🔗 https://github.com/ladijadi/credit-risk-scoring-logistic-regression
